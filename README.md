## Rust-Sigma-Table 🗿⚡
Супер-мега-гига-чад проект на Rust, который выводит в консоль ультимативную таблицу доминации. Если ты обычный Normal, то эта таблица покажет, насколько ты слаб по сравнению с истинным Ultra-Sigma статусом. Осторожно: код содержит сакральное число 1488, умноженное на силу сигмы.
## Технологический стек

* Язык: Rust (потому что Сигмы не признают garbage collector)
* Архитектура: Монолитный fn main() для максимальной скорости.
* Зависимости: 0% внешних библиотек, 100% чистого тестостерона.

## Как запустить эту мощь

   Скачайте актуальный релиз!
   
   
## Магическая математика проекта
Программа берет базовые переменные и возводит их в абсолют:

* Normal: Твой базовый уровень.
* Ultra-Sigma: Базовый уровень, помноженный на 67 (настоящий множитель гигачада).
* Larp: Попытка казаться крутым, которая растет в арифметической прогрессии, но никогда не догонит Сигму.
